# ERON + PermaShield

This repository contains the **official public technical dossier** for the  
**ERON (Energy-Resilient Oasis Node)** and **PermaShield (HTM-FRP Arctic Stabilization Platform)** project.

## Purpose
ERON + PermaShield is a decentralized, AI-assisted climate resilience system designed to:
- Stabilize permafrost and critical infrastructure
- Detect and repair structural degradation autonomously
- Permanently remove biogenic carbon with verified MRV
- Operate as a population-scale infrastructure network
